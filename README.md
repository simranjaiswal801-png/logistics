# Logistics Accounting Front-End Prototype

This project is a **front-end prototype** of a **logistics-focused accounting system**, customized from the HelloBooks MVP.  
It is designed specifically for **logistics and transportation businesses** to manage shipments, invoices, expenses, vehicles, and reports in a more industry-relevant way.
